## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/librepathai/librepath
cd librepath
```

2. Install dependencies:

```bash
bun install
```

3. Run the development server:

```bash
bun dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.
